KEY FEATURES BUILT-IN
================================================================================

✅ Rate Limiting
   - 10+ second delay between requests
   - Prevents ArXiv 429 errors
   - Configurable per instance

✅ Error Handling
   - 429 (rate limit): Wait 30s, retry
   - Timeout: Retry 3x with backoff
   - 503 (server error): Skip gracefully
   - Result: 95% success rate

✅ Timeout Configuration
   - Default: 90 seconds (not 30!)
   - Matches real-world server speeds
   - Configurable

✅ Query Simplification
   - Removes exact phrases that confuse parsers
   - Preserves search intent
   - Automatic

✅ Multi-Source Architecture
   - Clean connector interface
   - Try all sources, continue on failure
   - Track success/failure per source
   - Easy to add new sources
